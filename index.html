SET MODE
Press OPTIONS to change Team Names and system preferences.
Operation Guide
SET — arm a correction; then press the field key (TIME, PERIOD, HOME SCORE, GUEST SCORE, GOAL SHOTS, GOAL SAVES) and enter digits + ENTER
TIME ON / TIME OFF — start / stop the game clock
NEW MINOR / NEW MAJOR — enqueue 2:00 / 5:00 penalty; type player # + ENTER
INSERT PENALTY — press a team key (HOME / GUESTS), then player # ENTER, then MMSS / MSS ENTER
PENALTY ON / OFF — pause / resume penalty countdown
CLEAR PENALTY — press a team key, then 1 or 2 to clear that slot
HOME SCORE / GUEST SCORE — +1 goal (lights GOAL light); in SET, type digits + ENTER
GOAL SHOTS / GOAL SAVES — +1 stat; in SET, enter exact value
PERIOD — +1 period (resets clock); in SET, enter exact value
HORN (H) — manual horn while held
OPTIONS / BLANK / SCROLL PROFILES / TIME OF DAY — stub indicators
YES / NO — respond to confirm prompts
