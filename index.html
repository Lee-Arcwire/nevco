Operation Guide
SPACE — toggle SET / RUN
H — manual horn (hold)
RUN — SCORE / SHOTS / TOL buttons increment (decrement in SET)
SET — SCORE / SHOTS / TOL / PERIOD buttons begin numeric entry (type digits, ENTER)
CLOCK SET — type MMSS or MSS, ENTER (clock must be stopped)
PENALTY — type player (1-2 digits) ENTER, then duration MMSS or MSS, ENTER
AUTO HORN — toggles automatic horn when clock reaches 0
LAMP TEST — lights all segments for 3 seconds
RESET — press while in SET mode to reset the game
